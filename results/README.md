# Results Directory

This directory stores lightweight result artifacts generated during the DeepFormer refactored demo workflow.

## Intended contents

Examples include:

- `deepsea_mat_summary.txt`
- `deepformer_demo_train_log.txt`
- `deepformer_demo_result_summary.txt`
- `deepformer_demo_eval_log.txt`

## Large files

Large model checkpoints and large runtime artifacts should generally not be committed to GitHub unless they are intentionally kept for demonstration purposes and remain small enough to be manageable.

## Current role

In the current project stage, this directory is mainly used to preserve:

- MAT-route verification summaries;
- tiny demo training logs;
- tiny demo evaluation logs;
- concise result summaries for project presentation and defense.
