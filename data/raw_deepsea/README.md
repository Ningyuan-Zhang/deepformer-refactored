# raw_deepsea

This directory stores the reconstructed DeepSEA public-data outputs:

- `train.mat`
- `valid.mat`
- `test.mat`

These files were generated from publicly available DeepSEA-related source files and the hg19 reference genome using a public reconstruction workflow.

They are treated as large runtime artifacts and are not intended to be committed to GitHub.

Their role in this project is to document and preserve the public-data reconstruction result for future runtime adaptation and demonstration.
